The `NVIDIA Developer Tools <https://developer.nvidia.com/tools-overview>`__
are a collection of applications, spanning desktop and mobile targets, which
enable developers to build, debug, profile, and develop class-leading and
cutting-edge software that utilizes the latest visual computing hardware from
NVIDIA.

The performance tools from the `VI-HPS <http://www.vi-hps.org/tools/>`__
Institute are also supported on NVIDIA gpus.
